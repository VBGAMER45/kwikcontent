Should You Upgrade To Version 1.2?

It's not necessary to upgrade existing domains to the latest version of Kwikcontent. An upgrade is only recommended if you would like to take advantage of the new Amazon and eBay integration feature.

The SEO URL feature is useful for new installations only. SEO URLs result in duplicate content for existing installations as the old URLs will continue to function as well. If you do upgrade an existing Kwikcontent installation that already has some pages indexed in the search engines, you should turn off SEO URLs (see below).

Note: Any manual modifications you might have made to Kwikcontent's PHP or HTML files will be lost, as they will be overwritten by the upgrade.

How To Upgrade

  1. Backup your existing installation of Kwikcontent so you can revert to it if something goes wrong during the upgrade process.
  2. Download the Zip file which contains your domain's Kwikcontent application.
  3. Extract the Kwikcontent Zip file to your local hard drive. (It's best to extract the file twice: Once for any new installations you might want to make in the future, and again -- to a different directory -- for upgrades. You'll be making a few changes to the upgrade files below, so it's useful to keep the two extractions separate.)
  4. Import the file upgrade-to-1.2.sql (found in your Kwikcontent /install directory) into your existing Kwikcontent MySQL database. (Get more information about this step)
  5. In the /includes directory, delete the file _functions.php and then rename the file _functions-oldurls.php to _functions.php. This ensures backwards compatibility with the old-style URLs.
  6. If you renamed your /admin directory on your original installation, rename the /admin directory you just extracted to that same name.
  7. Upload all of the files to the web root directory of your domain (except for the /install directory). Overwrite everything.
  8. Visit the admin panel of your Kwikcontent software (www.yourdomain.com/admin) and click into the Settings area. You will need to enter your the following values:
    • Domain Title - The name of your website; this should be a short and powerful keyphrase, without the domain extension
    • Amazon Affiliate ID - You can get this from Amazon Associates.
    • Amazon Global Keyphrase - Enter one or two words relevant to your site here. Be as broad as possible to get the best results. You might have to experiment a little until your site displays suitable results.
    • eBay CampaignID - You can get this from eBay Partner Network. First create an account, then create a campaign and enter the CampaignID here.
    • eBay Front Page Keyphrase - Enter one or two words relevant to your site here. These words are used only for eBay ads on the front page of your domain. All other eBay ads are automatically matched to the category they are displayed in.

Your upgrade is complete. You can now begin updating your site's content. If you have any problems (or would like to share your suggestions for improving Kwikcontent) please visit our support forum.
