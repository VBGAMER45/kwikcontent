Kwikcontent v1.2 Installation Instructions

(If you'd like to upgrade an existing installation of Kwikcontent, read this file instead.)

Make sure you have read the System Requirements before you attempt to install Kwikcontent on your server. Currently, there is no web-based installer for Kwikcontent available. However, these step-by-step instructions should prove very easy for most users.

If you are not comfortable installing your own Kwikcontent software, we can assist you for a small fee. You can purchase a Kwikcontent installation here. You can also purchase multiple installations at a discount.

  1. Download the Zip file which contains your domain's Kwikcontent application.
  2. Extract the Kwikcontent Zip file to your local hard drive.
  3. Create a new MySQL database on your domain.
  4. Import the db.sql file (found in your Kwikcontent /install directory) into the MySQL database you just created. (Get more information about this step)
  5. Open the _global.php.rename file found in the /includes directory. Update the MySQL database variables with your proper MySQL database information. Also update the variables associated with the domain you are installing on. Although it is not necessary, advanced users can also update other defined variables in this file. Save the file as _global.php (or save it first, then rename it to _global.php)
  6. Upload all of the files to the web root directory of your domain (except for the /install directory).
  7. Password protect your /admin directory. Note: This directory can be renamed if desired.
  8. CHMOD the following directories to 0777:
    • /cache
    • /templates_c
    • /settings
  9. Visit the admin panel of your Kwikcontent software (www.yourdomain.com/admin) and click into the Settings area. You will need to save your to enter two api keys:
    Yahoo! Application ID in this section before you can begin to update your site content. If you do not yet have a Yahoo application ID, you can obtain one (free of charge) here. Bing Application ID is needd as well. You can get a free api key from http://www.bing.com/developers/createapp.aspx
  10. Most of the other fields are self-explanatory, but these deserve special attention:
    • Domain Title - The name of your website; this should be a short and powerful keyphrase, without the domain extension
    • Amazon Affiliate ID - You can get this from Amazon Associates.
    • Amazon Global Keyphrase - Enter one or two words relevant to your site here. Be as broad as possible to get the best results. You might have to experiment a little until your site displays suitable results.
    • eBay CampaignID - You can get this from eBay Partner Network. First create an account, then create a campaign and enter the CampaignID here.
    • eBay Front Page Keyphrase - Enter one or two words relevant to your site here. These words are used only for eBay ads on the front page of your domain. All other eBay ads are automatically matched to the category they are displayed in.

Your installation is complete. You can now begin updating your site's content. If you have any problems (or would like to share your suggestions for improving Kwikcontent) please watch the Kwikcontent Demo Video and/or visit our support forum.

Troubleshooting

If your site appears to work but no matter which link you click on, you are taken back to the front page, then you did not upload the .htaccess file to your web root directory. This is a common oversight because many FTP applications, in their default configuration, simply hide files that start with a dot. For your convenience we have included a file named htaccess.txt; simply rename this file to .htaccess on your sever and your site should start to work.
